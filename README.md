# AI 면접 코치

> 이력서 + 채용공고 기반 AI 맞춤 모의면접 서비스

이력서와 채용공고를 분석해 개인화된 면접 질문을 생성하고, 3가지 유형의 AI 면접관과 실전 모의면접을 진행합니다. 면접 종료 후 면접관 간 토론 과정과 항목별 피드백을 제공합니다.

## 핵심 기능

- 채용공고 URL 입력만으로 직무 분석 및 맞춤 질문 자동 생성
- 조직 적합성 · 논리력 · 기술 역량 3가지 관점의 AI 면접관
- 면접관 간 토론 과정 시각화 및 구체적 개선 포인트 제공

## 기술 스택

| 영역 | 기술 |
|------|------|
| 프레임워크 | Next.js 14 (App Router) + TypeScript |
| 인증 · DB | Supabase Auth + PostgreSQL |
| 크롤링 | Jina Reader API |
| LLM | Ollama (자체 호스팅) |
| 스타일링 | Tailwind CSS |
| 검증 | Zod |
